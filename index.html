关于 Outbound

《Outbound》是一款背景设定在近未来、以舒适房车为主题的开放世界生存游戏。你将从一辆空荡荡的房车开始，逐步将其打造成你的梦想家园。 你可以利用太阳能发电，建立自动化生产线，并在不断变化的世界中探索。

PC 系统要求

最低配置

  • OS: Windows 10/11 (64-bit)
  • 处理器: Intel Core i5-4590
  • 内存: 8 GB RAM
  • 显卡: GTX 1050 / RX 560
  • 存储: 10 GB 可用空间

推荐配置

  • OS: Windows 10/11 (64-bit)
  • 处理器: Intel Core i7-8700K
  • 内存: 16 GB RAM
  • 显卡: RTX 2060 / RX 5700

新手入门指南

⚡ 能源管理

优先安装车顶太阳能电池板，这是你所有设备的动力来源。

🛠️ 资源采集

砍伐树木和采集矿石，利用初始工作台制作基础建材。

🚐 扩建策略

房车内部空间有限，合理布局储物柜可以极大提高效率。
